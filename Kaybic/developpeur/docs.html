Documentation API

Guide complet pour intégrer les APIs de paiement mobile KAYBIC AFRICA

Version 2.1 REST API JSON

Introduction

L'API KAYBIC AFRICA est une API REST qui vous permet d'intégrer facilement les services de paiement mobile dans vos applications. Notre API prend en charge tous les principaux opérateurs de mobile money en Afrique francophone.

Rapide

Transactions en temps réel

Sécurisé

Chiffrement SSL/TLS

Étendu

15+ pays africains

URL de base

https://api.kaybic.africa/v2

Authentification

L'API KAYBIC AFRICA utilise des clés API pour l'authentification. Vous devez inclure votre clé API dans l'en-tête de chaque requête.

En-têtes requis

Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
Accept: application/json
Important : Gardez vos clés API secrètes. Ne les exposez jamais dans le code frontend.

Environnements

Environnement URL de base Description
Sandbox https://sandbox-api.kaybic.africa/v2 Pour les tests et le développement
Production https://api.kaybic.africa/v2 Pour les transactions réelles

Démarrage Rapide

Voici un exemple simple pour effectuer votre première transaction avec notre API.

1. Obtenir vos clés API

  1. Créez un compte sur notre plateforme
  2. Validez votre compte avec les documents KYC
  3. Récupérez vos clés API dans le tableau de bord

2. Faire un appel API

Exemple d'encaissement avec PayIn API :

    "reference": "TXN_123456789"
  }'

PayIn API - Encaissement

L'API PayIn vous permet de collecter des paiements depuis les portefeuilles mobile money de vos clients.

Créer une transaction PayIn

POST /payin
Paramètres
Paramètre Type Requis Description
amount integer Oui Montant en centimes (ex: 1000 = 10.00)
currency string Oui Code devise ISO (XOF, XAF, GHS, etc.)
customer_phone string Oui Numéro de téléphone du client
provider string Oui Opérateur mobile money
reference string Oui Référence unique de transaction
Réponse
{
  "status": "success",
  "data": {
    "transaction_id": "txn_1234567890",
    "reference": "TXN_123456789",
    "amount": 1000,
    "currency": "XOF",
    "status": "pending",
    "provider": "orange_money_ci",
    "customer_phone": "+225123456789",
    "created_at": "2024-01-15T10:30:00Z"
  }
}

PayOut API - Décaissement

L'API PayOut vous permet d'envoyer de l'argent vers les portefeuilles mobile money de vos bénéficiaires.

Créer une transaction PayOut

POST /payout
Paramètres
Paramètre Type Requis Description
amount integer Oui Montant en centimes
currency string Oui Code devise ISO
recipient_phone string Oui Numéro du bénéficiaire
provider string Oui Opérateur mobile money
reference string Oui Référence unique

Transfer API - Transferts Internationaux

L'API Transfer permet les transferts d'argent entre différents pays africains.

Créer un transfert international

POST /transfer
Les transferts internationaux nécessitent une validation KYC renforcée.

Webhooks

Les webhooks vous permettent de recevoir des notifications en temps réel sur le statut de vos transactions.

Configuration

Configurez vos endpoints webhook dans votre tableau de bord. KAYBIC AFRICA enverra une requête POST à votre URL avec les détails de la transaction.

Événements

  • transaction.completed - Transaction terminée avec succès
  • transaction.failed - Transaction échouée
  • transaction.pending - Transaction en attente

Codes d'erreur

Code Message Description
400 Bad Request Paramètres manquants ou invalides
401 Unauthorized Clé API invalide ou manquante
429 Too Many Requests Limite de taux dépassée
500 Internal Server Error Erreur serveur interne

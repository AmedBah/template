Comment fonctionne notre API d'Encaissement

Collectez simplement les paiements de vos clients en 3 étapes faciles, quel que soit leur opérateur mobile money.

1

Initier le paiement

Envoyez une requête à notre API avec le montant, la devise et les détails du client. Nous générons une référence de paiement unique.

2

Client confirme

Le client reçoit une notification sur son téléphone et confirme le paiement en saisissant son PIN Mobile Money.

3

Confirmation instantanée

Vous recevez une confirmation en temps réel via webhook. Les fonds sont disponibles immédiatement sur votre compte KAYBIC.

Intégration simple avec notre API

Quelques lignes de code suffisent pour intégrer notre solution de paiement dans votre application.

// Initier un paiement avec KAYBIC AFRICA API
$api_key = "votre_api_key";
$api_secret = "votre_api_secret";

// Préparer les données de la requête
$payload = [
    "amount" => 5000,
    "currency" => "XOF",
    "phone_number" => "+22501234567",
    "operator" => "orange", // ou "mtn", "moov", "wave"
    "reference" => "INV-2025-001",
    "description" => "Paiement pour commande #12345",
    "callback_url" => "https://votresite.com/callback"
];

// Appel à l'API KAYBIC
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, "https://api.kaybic.africa/v1/collect");
curl_setopt($ch, CURLOPT_RETURNTRANSFER, TRUE);
curl_setopt($ch, CURLOPT_HEADER, FALSE);
curl_setopt($ch, CURLOPT_POST, TRUE);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($payload));
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    "Content-Type: application/json",
    "Authorization: Bearer " . $api_key,
    "X-API-Secret: " . $api_secret
]);

$response = curl_exec($ch);
curl_close($ch);

$result = json_decode($response);
echo "ID Transaction: " . $result->transaction_id;
# Initier un paiement avec KAYBIC AFRICA API
import requests
import json

api_key = "votre_api_key"
api_secret = "votre_api_secret"

# Préparer les données de la requête
payload = {
    "amount": 5000,
    "currency": "XOF",
    "phone_number": "+22501234567",
    "operator": "orange", # ou "mtn", "moov", "wave"
    "reference": "INV-2025-001",
    "description": "Paiement pour commande #12345",
    "callback_url": "https://votresite.com/callback"
}

# Appel à l'API KAYBIC
headers = {
    "Content-Type": "application/json",
    "Authorization": f"Bearer {api_key}",
    "X-API-Secret": api_secret
}

response = requests.post(
    "https://api.kaybic.africa/v1/collect",
    data=json.dumps(payload),
    headers=headers
)

result = response.json()
print(f"ID Transaction: {result['transaction_id']}")
// Initier un paiement avec KAYBIC AFRICA API
const axios = require('axios');

const api_key = "votre_api_key";
const api_secret = "votre_api_secret";

// Préparer les données de la requête
const payload = {
    amount: 5000,
    currency: "XOF",
    phone_number: "+22501234567",
    operator: "orange", // ou "mtn", "moov", "wave"
    reference: "INV-2025-001",
    description: "Paiement pour commande #12345",
    callback_url: "https://votresite.com/callback"
};

// Appel à l'API KAYBIC
axios.post('https://api.kaybic.africa/v1/collect', payload, {
    headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${api_key}`,
        'X-API-Secret': api_secret
    }
})
.then(response => {
    console.log(`ID Transaction: ${response.data.transaction_id}`);
})
.catch(error => {
    console.error('Erreur:', error.response.data);
});

Tout ce que vous devez savoir sur notre API d'Encaissement Multi-Opérateurs.

L'intégration de notre API d'Encaissement est particulièrement simple et rapide. La plupart de nos clients sont opérationnels en moins de 48 heures. Notre équipe technique vous accompagne durant tout le processus pour garantir une mise en place sans accroc.

Notre API supporte tous les principaux opérateurs Mobile Money en Afrique de l'Ouest : Orange Money, MTN Mobile Money, Moov Money, Wave, Wizall Money, ainsi que les solutions de paiement des banques locales. Nous couvrons 8 pays de la CEDEAO : Côte d'Ivoire, Sénégal, Mali, Burkina Faso, Bénin, Togo, Niger et Guinée.

Nous vous notifions des paiements de trois façons : via des webhooks en temps réel envoyés à votre serveur, par email, et dans votre tableau de bord KAYBIC. Vous pouvez configurer les notifications selon vos besoins spécifiques. Les webhooks incluent toutes les informations de transaction nécessaires pour mettre à jour vos systèmes automatiquement.

La sécurité est notre priorité. Nous utilisons un cryptage SSL/TLS pour toutes les communications, l'authentification API par clés, la vérification de signature des webhooks, et nous suivons les normes PCI-DSS. Nos systèmes sont régulièrement audités par des experts en sécurité indépendants, et nous effectuons des tests d'intrusion périodiques pour garantir la protection de vos données et de vos fonds.

Vous pouvez retirer vos fonds à tout moment via notre interface de gestion. Nous proposons des règlements quotidiens, hebdomadaires ou mensuels selon votre choix, directement sur votre compte bancaire ou mobile money. Les fonds sont généralement disponibles sous 24h pour les transferts bancaires et instantanément pour les transferts mobile money.

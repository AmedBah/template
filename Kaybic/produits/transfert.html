Intégration API

Intégrez nos services de transfert en quelques lignes de code

<?php
$curl = curl_init();

curl_setopt_array($curl, array(
  CURLOPT_URL => 'https://api.kaybic.africa/v1/international-transfer',
  CURLOPT_RETURNTRANSFER => true,
  CURLOPT_ENCODING => '',
  CURLOPT_MAXREDIRS => 10,
  CURLOPT_TIMEOUT => 0,
  CURLOPT_FOLLOWLOCATION => true,
  CURLOPT_HTTP_VERSION => CURL_HTTP_VERSION_1_1,
  CURLOPT_CUSTOMREQUEST => 'POST',
  CURLOPT_POSTFIELDS => json_encode([
    'sender' => [
      'name' => 'Jean Dupont',
      'phone' => '+22670123456',
      'country' => 'CI'
    ],
    'recipient' => [
      'name' => 'Marie Martin',
      'phone' => '+33612345678',
      'country' => 'FR',
      'bank_account' => 'FR1420041010050500013M02606'
    ],
    'amount' => 100000,
    'currency_from' => 'XOF',
    'currency_to' => 'EUR',
    'purpose' => 'family_support'
  ]),
  CURLOPT_HTTPHEADER => array(
    'Authorization: Bearer YOUR_API_KEY',
    'Content-Type: application/json'
  ),
));

$response = curl_exec($curl);
curl_close($curl);

$data = json_decode($response, true);
echo "Transaction ID: " . $data['transaction_id'];
?>

Questions fréquentes

Tout ce que vous devez savoir sur nos transferts internationaux

Les délais varient selon la destination : instantané pour l'Afrique francophone, moins de 15 minutes pour l'Europe et l'Amérique du Nord, et jusqu'à 30 minutes pour l'Asie. La plupart des transferts sont effectués en moins de 5 minutes.

Les limites dépendent du niveau de vérification du compte : 500 000 FCFA/jour pour les comptes de base, 2 000 000 FCFA/jour pour les comptes vérifiés, et jusqu'à 10 000 000 FCFA/jour pour les comptes entreprise.

Nous utilisons les taux interbancaires en temps réel avec une marge de 0.5% seulement. Les taux sont mis à jour toutes les minutes pour garantir la transparence et la compétitivité.

Nous avons besoin des informations complètes de l'expéditeur et du bénéficiaire (nom, téléphone, pays), du montant, des devises source et destination, et du motif du transfert pour la conformité réglementaire.

Chaque transfert génère un ID unique que vous pouvez utiliser avec notre API de suivi. Vous recevrez également des webhooks automatiques à chaque étape du processus (initiation, validation, exécution, réception).

Paiements

KAYBIC AFRICA accepte tous les principaux moyens de paiement mobile en Afrique francophone :

  • Mobile Money : Orange Money, MTN Mobile Money, Moov Money, Wave
  • Cartes bancaires : Visa, Mastercard, cartes locales
  • Portefeuilles électroniques : PayPal, Payoneer (selon les pays)
  • Virements bancaires : Transferts directs depuis les comptes bancaires

La disponibilité peut varier selon votre pays. Consultez notre documentation pour plus de détails.

Les délais de traitement varient selon le type de transaction :

  • Encaissement (PayIn) : Instantané à 2 minutes
  • Décaissement (PayOut) : 5 minutes à 2 heures
  • Transferts internationaux : 1 à 24 heures selon la destination

La plupart des transactions sont traitées en temps réel grâce à notre infrastructure optimisée.

En cas d'échec de paiement :

  1. Vérifiez le statut de la transaction dans votre tableau de bord
  2. Consultez le code d'erreur pour identifier la cause
  3. Vérifiez que le compte destinataire a suffisamment de fonds
  4. Contactez notre support avec l'ID de transaction

Notre système de retry automatique tente de reprendre les transactions échouées lorsque c'est possible.

API & Intégration

Pour commencer l'intégration :

  1. Créez un compte : Inscrivez-vous sur notre plateforme
  2. Obtenez vos clés API : Récupérez vos clés de test et de production
  3. Consultez la documentation : Lisez notre guide d'intégration détaillé
  4. Testez en sandbox : Utilisez notre environnement de test
  5. Passez en production : Une fois les tests validés

Notre équipe technique est disponible pour vous accompagner à chaque étape.

Notre API REST est compatible avec tous les langages de programmation. Nous fournissons des SDK officiels pour :

  • PHP : SDK complet avec exemples
  • Node.js : Package NPM disponible
  • Python : Bibliothèque Python pip
  • Java : Librairie Maven

Des exemples de code sont disponibles pour d'autres langages comme C#, Ruby, Go dans notre documentation.

Les limites dépendent de votre plan d'abonnement :

  • Plan Starter : 1000 appels/heure
  • Plan Business : 10 000 appels/heure
  • Plan Enterprise : Limites personnalisées

Si vous atteignez les limites, nous retournons un code d'erreur 429. Vous pouvez upgrader votre plan à tout moment.

Sécurité

Nous utilisons les plus hauts standards de sécurité :

  • Chiffrement SSL/TLS : Toutes les communications sont chiffrées
  • Tokenisation : Les données sensibles sont tokenisées
  • Conformité PCI DSS : Certification niveau 1
  • Authentification 2FA : Double authentification requise
  • Monitoring 24/7 : Surveillance continue des systèmes

Vos données financières ne sont jamais stockées en clair sur nos serveurs.

Bonnes pratiques pour sécuriser vos clés API :

  • Ne jamais exposer les clés dans le code frontend
  • Utiliser des variables d'environnement
  • Renouveler régulièrement vos clés
  • Configurer les restrictions IP
  • Utiliser des webhooks sécurisés

Notre tableau de bord vous permet de gérer et surveiller l'usage de vos clés API.

Compte

Créer un compte est simple et rapide :

  1. Cliquez sur "Commencer" sur notre page d'accueil
  2. Remplissez le formulaire d'inscription
  3. Vérifiez votre email
  4. Complétez votre profil
  5. Soumettez vos documents KYC

L'activation du compte prend généralement 24 à 48 heures après validation KYC.

Pour modifier votre plan :

  1. Connectez-vous à votre tableau de bord
  2. Allez dans "Paramètres" > "Abonnement"
  3. Sélectionnez votre nouveau plan
  4. Confirmez les changements

Les upgrades sont effectifs immédiatement. Les downgrades prennent effet au prochain cycle de facturation.
